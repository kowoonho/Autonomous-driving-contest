1. 실행하는법

    Main.py
        rf_weight_file을 원하는 weight_file로 지정
    
    road_following 폴더에서
        python Main.py

2. Data augmentation

    Dataset/augmentation.py
        1) file_path를 수집한 데이터의 디렉토리로 지정
        2) new_dir_path를 augmentation한 데이터가 들어갈 디렉토리로 지정
        * new_dir_path에는 원래 데이터는 들어가지 않고, augmentation한 데이터만 들어가니 주의
            * new_dir_path에 augmentation한 데이터를 넣어두고, 원래 데이터와 합칠때 옮기는걸 추천합니다.
    
    road_following/Dataset 폴더에서
        python augmentation.py

3. Train
    train.py
        1) BEST_MODEL_PATH를 원하는 weight_file 이름으로 지정
            ex) best_steering_model_0114a.pth
        2) rfdataset_path를 training시킬 데이터가 있는 디렉토리로 지정
        
    * 서버에서 실행시 주의할점
        서버에서 nvidia-smi를 입력해서 미사용중인 gpu번호를 확인하고,
        train.py의 device = "cuda:{gpu번호}" if torch.cuda.is_available() else "cpu"
        위와 같이 변경 후 사용할 것

    road_following 폴더에서
        python train.py

4. Prediction
    predict.py
        1) test_file_dir_path를 test할 데이터가 있는 디렉토리로 지정
        2) weight_file을 training한 weight_file로 지정
            * test_data는 몇개 따로 수집해야함
    road_following 폴더에서
        python predict.py

5. steering_model weight file
    steering_model_info.txt에 어떤 데이터로 학습시킨 파일인지 써놨습니다.
